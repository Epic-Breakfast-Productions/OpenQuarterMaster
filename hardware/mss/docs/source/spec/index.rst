MSS Module Communication Spec
#############################

This section is concerned with the formal description of the MSS module communication specification.

To facilitate the operations of managing and utilizing the hardware modules from the controller, we define this communication
protocol. It is defined openly and generically here in order to help drive adoption and improvements.
