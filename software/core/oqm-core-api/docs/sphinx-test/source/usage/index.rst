Usage / User Guide
#####################

This section is concerned with how to actually use the service. It is mainly concerned with conceptual usage, more than
providing concrete examples of code to call in specific languages, though data examples are generally provided where
useful.

.. note::

   For concrete, and specific documentation on individual calls and how to make them against the API, open the swagger
   documentation on your instance. This can be found at the ``/q/swagger-ui`` endpoint. Example, the swagger docs on our
   `demo <https://openquartermaster.com/public/demo>`_ instance: https://demo.openquartermaster.com/core/api/q/swagger-ui
